Cara Penggunaan
1. Melihat Daftar Pasien
Buka aplikasi, secara default akan menampilkan daftar pasien , Gunakan search bar untuk mencari pasien berdasarkan nama atau NIK, Click tombol "Nama" atau "Tanggal" untuk sorting

2. Menambah Pasien Baru
Click tombol "Tambah Pasien" di pojok kanan atas, Isi formulir dengan data pasien:
Nama Lengkap (min 3 karakter)
NIK (16 digit angka)
Diagnosa Masuk
Tanggal Masuk
Dokter Penanggung Jawab
Ruangan

3. Validasi Form
Field yang kosong akan menampilkan error
